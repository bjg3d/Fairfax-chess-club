# Fairfax Chess Club website

This is a static, GitHub Pages-ready website for Fairfax Chess Club.

## Files

- `index.html`: the full website, including embedded CSS and a tiny script for the footer year.
- `assets/logo.svg`: club logo.
- `assets/hero-board.svg`: homepage hero illustration.
- `assets/board-accent.svg`: decorative accent used in the join section.
- `assets/favicon.svg`: browser tab icon.
- `assets/social-preview.svg`: sharing preview graphic.

## How to publish on GitHub Pages

1. Create a new GitHub repository, for example `fairfax-chess-club`.
2. Upload `index.html` and the `assets` folder to the repository root.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder, then save.
6. GitHub will publish the website at a URL like `https://YOUR-USERNAME.github.io/fairfax-chess-club/`.

## What to edit first

Search in `index.html` for `EDIT THIS`. Those comments mark the places most likely to need real club details:

- Meeting day and time
- Venue name and address
- Weekly announcement
- Cost or dues
- Scholastic policy
- Contact email

Replace every instance of `fairfaxchessclub@example.com` with the real club email address.
